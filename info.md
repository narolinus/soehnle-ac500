# Soehnle AC500

Home Assistant custom integration for the Soehnle Airclean Connect 500 over Bluetooth.

Features:

- setup directly from Home Assistant
- pairing via a dedicated Home Assistant action
- support for connectable Bluetooth proxies
- separate status polling and short-lived control-mode sessions
- fan, timer, UV-C, auto mode, night mode, buzzer, PM2.5, temperature and filter status
